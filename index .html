HTML Começaremos falando sobre o HTML, uma linguagem de marcação chama-da em inglês HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. Esta linguagem tem como principal objetivo auxiliar na definição da estrutura de documentos (páginas) Web. Tim Berners-Lee criou o HTML com a ideia de suprir a necessidade de criar documentos des-te tipo. Em torno dos anos 1990, quando Marc Andreessen criou o navegador Mosaice ele foi 22muito utilizado, o HTML ganhou força e foi adotado por outros desenvolvedores e fabricante de navegadores, compartilhando as mesmas convenções. A composição da linguagem HTML é feita por meio de TAGS que são utilizadas no começo e no final do texto que será afetado pelo código. Com a utilização desta linguagem, é possível criar conteúdos, como texto, ima-gens, vídeos, áudio, entre outros, e depois publicá-los na internet. Conforme o tempo foi passando, nasceram outras versões do HTML: • HTML+; • HTML 2.0; • HTML 3. Agora vamor exercitar! Crie e publique uma página utilizando elementos HTML. HTML Iframes-iFrame pode ser adaptado do inglês como quadro, estrutura, composição, trecho ou formação. No contexto da computação, o iFrame é um trecho de código HTML que pode ser inserido em qualquer parte de um site. Com ele, o usuário consegue incorporar conteúdos de diferentes formatos e estilos É possível inserir documentos para baixar, fotos, vídeos, imagens e sons. Além de outros tipos de mídias interativas, como games e questionários com opções de escolha. Essa integração de conteúdo acontece através do simples posicionamento de um código já formatado e disponível em outras fontes. É possível posicionar o conteúdo onde o usuário achar melhor sem ter que realmente colocá-lo na estrutura geral de layout da página.
